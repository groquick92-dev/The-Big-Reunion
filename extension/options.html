🏡 The Big Reunion — Paramètres

Configurez la connexion à votre serveur Flask.

URL complète de votre application Flask (ex: https://votre-app.onrender.com)
Clé affichée dans la console Flask au démarrage. Pour la rendre permanente, définissez la variable d'environnement MANUAL_API_KEY sur votre serveur.
Comment récupérer votre clé API ?

1. Lancez votre serveur Flask : python app.py
2. Dans la console, cherchez le bloc entouré de ===
3. Copiez la clé générée et collez-la ci-dessus.

Pour une clé permanente (recommandé si déployé en ligne) :
Définissez MANUAL_API_KEY=votre_cle dans les variables d'environnement de votre hébergeur (Render, Railway, etc.) avant de lancer l'app.
